Nomad - Nord Modular Editor
---------------------------

Starting Nomad
--------------

Nomad can by started by the following command:

	java -jar nomad.jar

Nomad - Version 0.2.1 (Current)
-------------------------------

- saving a patch to a file might result in loss of information
- not all data is transmitted when downloading a patch to the synthesizer
